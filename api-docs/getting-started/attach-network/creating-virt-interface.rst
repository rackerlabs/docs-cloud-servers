Creating a virtual interface
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

When you create a virtual interface, you specify the following information:

- ``Network ID``. The ID of the network for which you want to create a virtual
  interface.
- ``Server ID``. The id of the server instance to which you want to connect the
  virtual interface.
