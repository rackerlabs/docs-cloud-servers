Migrate server
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^


Migrates a server.

**Usage:**

.. code::  

    $ nova migrate <server>

**Positional argument:**

-  ``server``. The name or ID of the server that you want to migrate.

**Optional argument:**

-  **--poll**. Blocks while the instance migrates so progress can be reported.

**Output:** None.
