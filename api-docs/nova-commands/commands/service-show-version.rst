Show service version
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Shows the version for a specified service.

**Usage:**

.. code::  

    nova service-version <id>

**Positional argument:**

-  ``id``. The ID of the service for which you want to show version information.

**Output:**

.. code::  

    2012.1-dev
