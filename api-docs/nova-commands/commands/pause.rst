Pause server
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Pauses a server.

**Usage:**

.. code::  

    $ nova pause <server>

**Positional argument:**

-  ``server``. The name or ID of the server that you want to pause.

**Optional argument:**

-  **--poll**. Blocks while the instance pauses so progress can be reported.

**Output:** None.
